# third-part-jar

> 对接第三方时，有些厂商直接给你甩一个sdk压缩包。maven依赖都不给。多人开发时，经常出现这些依赖不存在或者需要独自安装的方式。增加些许工作量。
>
> 最终采用maven上传sdk并将依赖安装到jitpack本地。然后使用jitpack去拉取依赖。
